# 探索伙伴 (ExploraPal) - 多模态AI学习平台

## 项目简介

探索伙伴是一个专为儿童设计的多模态AI学习平台，采用"从教知识点转向辅助探索"的理念，帮助孩子通过多模态AI技术，将兴趣爱好转化为深度研究项目。

## 核心理念

- **从填充式学习到探索式学习**：不直接灌输知识，而是通过AI引导孩子自主探索
- **兴趣驱动**：从孩子的真实兴趣出发（如恐龙、火箭、Minecraft）
- **多模态交互**：支持图像识别、语音处理、视频分析、AI创作
- **项目式学习**：将探索过程组织为完整的"研究项目"

## 核心功能

### 🎯 多模态AI学习平台 ✅ 已完全实现

#### 完整学习流程：观察 → 提问 → 表达 → 创作

1. **观察阶段** 🔍
   - 孩子拍照上传恐龙图片
   - AI图像识别分析恐龙特征（qwen3-vl-plus）
   - 自动创建观察记录，支持无observation_id调用

2. **提问引导** ❓
   - 基于观察结果生成个性化问题（qwen-flash）
   - 问题类型：观察、推理、实验、比较
   - AI提供答案和延伸思考

3. **表达阶段** 🎤
   - 语音转文字记录想法（qwen3-omni-flash）
   - AI润色生成研究笔记（qwen-flash）
   - 文字转语音播报内容（qwen3-omni-flash）

4. **创作阶段** 🎬
   - 生成研究报告（qwen3-max）
   - 视频内容分析（qwen-vl-plus）
   - AI视频创作（qwen-vl-plus）

## 技术特色

- **🤖 双平台AI集成**: 集成阿里云Qwen系列 + 豆包Doubao大模型，支持视觉、语音、文本、视频处理
- **🏗️ 微服务架构**: 5个独立服务，确保高可用性和可扩展性
- **🎯 教育导向**: 专门为儿童学习场景优化，注重探索和创造力培养
- **🔧 企业级AI服务**: TAL MLOps + 星图AI双平台集成，保证稳定性和安全性
- **⚡ 智能调度**: 同步+异步AI能力，按任务类型自动路由到最适合的模型

## 🎪 快速体验

### 多模态AI演示
体验完整的AI学习流程，包括图像分析、语音处理、视频生成等功能：

```bash
# 启动完整演示服务
cd explorapal
./start_demo.sh

# 访问演示页面
# http://localhost:8080/demo.html
```

### 🎬 AI演示视频生成 (豆包Doubao)
基于完整的演示流程，自动生成专业的MP4演示视频：

```bash
# 生成演示视频（10秒，基于用户图片和描述）
cd explorapal
./生成演示视频.sh <图片文件路径> <润色后的描述文本>
```

**视频生成技术**:
- **AI平台**: 星图AI平台 (豆包Doubao)
- **AI模型**: doubao-seedance-1.0-lite-i2v (图像到视频生成)
- **输入图片**: 用户原始上传的照片 (base64编码后传入content)
- **文字描述**: 用户总结内容经过AI润色后的描述
- **视频时长**: 10秒 (豆包模型最大支持时长)
- **输出格式**: 高质量MP4教育视频
- **处理方式**: 异步API调用，5-10分钟处理时间

**使用示例**:
```bash
# 示例：基于恐龙图片生成学习视频
./生成演示视频.sh /path/to/dinosaur.jpg "小葫芦观察到了一只三角龙化石，上面有三只角和骨板，看起来非常威武。"

# 查询生成结果
./查询视频生成结果.sh <返回的任务ID>
```
